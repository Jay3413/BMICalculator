# frmBMI

## 簡短說明

`frmBMI` 是一個簡潔的 Windows Forms 應用程式，用於計算並顯示使用者的 BMI（身體質量指數）。

功能：
- 支援輸入身高（公分）與體重（公斤）。
- 按下「計算」可顯示 BMI 值與分類（過輕、正常、過重、肥胖）。
- 可清除輸入並重新計算。

技術與使用方式：
- 專案目標框架：.NET Framework 4.7.2。
- 在 Visual Studio 2022 開啟 `frmBMI.sln`，設定專案為啟動專案，按 F5 執行。

## 安裝與使用

1. 確保已安裝 .NET Framework 4.7.2 或更高版本。
2. 下載或克隆此專案到本地機器。
3. 使用 Visual Studio 2022 開啟 `frmBMI.sln`。
4. 設定專案為啟動專案。
5. 按 F5 鍵執行應用程式。
